学历要求：本科|工作经验：2年|公司性质：外资（非欧美）|公司规模：150-500人

1、建立、保持与媒体的良好关系，组织实施新闻发布活动，协调与企业附属公司宣传部门的关系；就创新方的品牌定位，项目内容及所需的市场推广主题活动，与旅行社销售代表进行沟通；    2、协助上级制定日常宣传的传播计划，撰写和发布官方新闻公告和媒体文章，监督媒体宣传效果；     3、协助制定各类公关活动方案，组织实施； 根据各类传播进行执行反馈，分析宣传效果；     4、文档的收集整理，建立公司宣传档案；    5、媒体接待、联络及回复等；    6、协助市场部的策划、推广工作的方案制定、落地实施等；    7、协助上级完成其他市场相关的任务。任职要求：1、具备良好的写作能力，能针对市场推广需求，撰写创新方品牌及旗下项目产品相关的新闻稿；2、思维敏捷，富创意、有见解、具备一定的市场洞察能力，能从旅游、商业、消费、娱乐等角度进行新闻话题策划；3、善于沟通，具有良好的人际交往和协调能力、抗压能力，能协助建立与维护媒体关系，日常沟通、对接，为公司发展取得媒体支持，营造良好的舆论环境；4、全日制本科以上学历，市场营销、新闻传播、广告学、中文等相关专业；5、能够熟练运用PPT、word、excel等办公软件，能够熟练使用PPT制作策划方案；6、具有良好的英文听说读写能力；7、3年以上企业市场部门或策划代理公司相关经验。有从事新闻媒体、宣传报道等工作者优先考虑； 8、身体健康、征信记录良好、具备良好的职业操守。
