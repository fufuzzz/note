岗位要求： 1.本科以上学历，光学相关专业，2年以上相关工作经验；几何光学：精通应用成像光学、非成像照明光学，掌握激光原理、光学测量、光学设计等专业知识，具备扎实的几何光学理论，能够熟练运用相关光学设计软件。熟悉精密光学仪器设计、加工工艺及装配工艺文件的编写；具有光学系统设计能力和设计经验。熟悉光学冷加工工艺、光机系统装配工艺、光学参数检测原理、方法和仪器。光学元件采购、光纤放大器搭建、测试，超快实验系统设计，实验结果总结，分析，文档整理、方案调研；物理光学：熟悉光学仿真及设计，能熟练使用光学模拟软件。了解激光、激光加工原理，熟悉光路传输、光学元器件选型、光束处理等光路系统设计问题。能够独立设计实验方案、独立分析处理实验数据；；工作职责：1.负责光学系统设计与仿真、光学元件选型，光路调试；2.需要准确的预估光学设计和加工中可能出现的风险，并提出有效的规避措施；3.制定用于生产、装配、调试、测量用的相关工艺；参与产品的研制过程；4.熟练使用熔接机、光谱仪、光束质量分析仪、自相关仪等光学测试设备；5.具有光纤激光器、固体激光器开发经验，.熟悉机械类绘图软件，熟练绘制光学系统平面图。熟练使用绘图软件、数据分析处理软件。
