岗位职责：1、根据公司要求，制定负责区域年度销售及费用计划，并组织区域实施、持续推进，保证月、年度销售目标的达成；2、负责区域内所有渠道客户的开发与维护，协调客户订单的有效配送，解决客户冲突，积极维护区域内客户关系；3、实施区域通路策略、价格策略、促销策略、市场推广策略，保证区域价格稳定、促销方案推广效果，市场推广策略的提升；4、组织开展月度区域促销方案，审批审核促销费用；5、不断提升区域终端网络推进，达成销售，管理区域市场；6、做好区域内下属员工的管理和指导工作；7、完成上级领导安排的其他工作任务。任职要求：1、工作地点：唐山。2、3年以上当地快消品从业经验（具有粮油业务经验者优先录用），具备一定的客户及渠道资源。3、熟悉当地永辉、乐天玛特、天客隆、京客隆系统，具有知名品牌区域管理工作经验。4、有现代KA渠道、商超渠道管理经验者优先。5、具备优秀的沟通协调能力和语言表达能力，公关、应变能力强，有良好的商务谈判技能。6、对工作有高度热情和责任心，自我控制能力强，能承受较大工作压力。7、具备一定的计算机操作基础，能够熟练使用Word/Excel/Powerpoint等办公软件，进行文字处理、数据分析、邮件收发及互联网操作等。有良好的文字表达能力。8、 身体健康，没有不良嗜好，有强烈的事业心和进取心，能适应出差工作。
