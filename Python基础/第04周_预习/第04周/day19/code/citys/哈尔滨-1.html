一、哈尔滨银行信用卡中心简介哈尔滨银行信用卡中心成立于2015年末，作为总行直属部门，承担哈尔滨银行信用卡的产品研发、营销推广、风险管理及日常运营等职能，是哈尔滨银行小额信贷战略的重要组成部分，是未来发展的核心部门和利润中心之一。经过三年的砥砺奋进，目前发卡量近95万张，资产规模达120亿。为进一步提升经营管理水平，推进业务高速发展，现面向社会招聘精英人士加入我们的行列，共赴新征程，再创新辉煌。     二、项目方简介    哈尔滨银行始终秉承“普惠金融，和谐共富”的经营理念，以实现“国内一流国际知名小额信贷银行”为核心战略目标，不断巩固在小额信贷领域的领先地位，提高综合竞争力。历经18年的发展，哈尔滨银行已在全国设立17家分行，在14个省及直辖市设立24家村镇银行，分支机构遍布全国六大行政区。    信用卡中心作为哈尔滨银行直属部门，承担哈尔滨银行信用卡的产品研发、营销推广、风险管理及日常运营等职能，是哈尔滨银行未来发展的核心部门和利润中心之一。2016年作为哈尔滨银行信用卡业务独立市场化运作的元年，信用卡将以“打造城商行***” 为战略目标，以客户为中心，以合规经营为基础，做好重点产品和特色业务的推广工作。    三、项目人员基本条件    （一）国民教育序列大学专科及以上学历；    （二）年龄35周岁及以下；    （三）健康状况良好；    （四）个人信用记录良好，无违规违纪违法行为；    （五）部分岗位条件优异者可适当放宽条件;    （六）职位类型：全职    （七）薪金待遇：基础工资+岗位工资+福利补助+绩效奖金；    （八）用工形式：项目外包 。    （九）工作地点：招聘岗位:哈尔滨银行信用卡分中心业务支持（催收岗）工作地点：哈尔滨市道里区西八道街二号薪酬：6000-8000元/月要求：了解国家金融法律法规及银行监管政策,熟悉银行金融产品相关知识；具有较强的语言表达、组织协调及沟通能。 职责：根据卡中心催收策略，对逾期账户进行催收工作；协助进行卡片风险管理，排查风险隐患。
